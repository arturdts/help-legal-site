# HELP – Site de Políticas (GitHub Pages)

Este repositório contém páginas estáticas exigidas para publicação do app nas stores:

- `/privacy/` – Política de Privacidade
- `/terms/` – Termos de Uso
- `/delete-account/` – Solicitar exclusão de conta
- `/support/` – Suporte

## Como publicar no GitHub Pages

1. Suba todos os arquivos para a branch `main`.
2. No GitHub: **Settings → Pages**.
3. Em **Build and deployment**:
   - **Source**: Deploy from a branch
   - **Branch**: `main` e **/(root)**
4. Aguarde a URL aparecer (1–3 minutos).
